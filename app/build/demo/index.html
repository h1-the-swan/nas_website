About the Nautilus Diagram

The center node represents the set of papers identified as science communication and misinformation papers. Surrounding nodes represent influential papers that have cited those papers. Larger nodes represent more influential papers (ranked by Eigenfactor score). Click on a node to view the paper (you may need to allow pop-ups for this).
